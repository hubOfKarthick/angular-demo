pipes

  • Transform data before display
  • Built in pipes include lowercase, uppercase, decimal, date, percent, currency etc
  • To apply a pipe on a bound property use the pipe character " | "
  • Pass parameters to pipe using colon " : "
  • Custom pipes can be created

    • {{name}}

    {{name | lowercase}}

    {{name | uppercase}}

    {{name | slice:4}}

    {{name | slice:3:6}}

    {{message | titlecase}}

    {{person | json}}

    {{3.14545 | number:'2.1-2'}}

    {{0.3 | percent}}

    {{0.3 | currency}}

    {{0.3 | currency:'INR'}}

    {{3 | currency:'INR':'code'}}

    {{date}}

    {{date | date:'short'}}

    {{date | date:'shortDate'}}

    {{date | date:'shortTime'}}
