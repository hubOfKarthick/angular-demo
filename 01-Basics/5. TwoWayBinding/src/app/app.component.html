  • Property Binding: This flows data from the component class to html element property.
  • Event binding : This flows data in the opposite direction i.e from the element to component class property.
  • Two-way data binding in Angular is a combination of both Property Binding and Event Binding.
  • ngModel directive - import FormsModule in appmodule.
