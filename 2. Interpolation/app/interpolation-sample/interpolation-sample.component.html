  • Expression that is enclosed in double curly braces is commonly called as Template Expresion
  • Template expression can be also an Ternary Operator

    • Thing can be done with Interpolation.

  • We can bind data from class to the template using the double curley braces.
  • We can Evaluate the javascript expression.
  • We can use javascript properties and methods also.
  • We can perform the string Concatenation.
  • We can call the class methods in the interpolation directly inside the expression.
  • We can concatenate a hard-coded string with the property value also
  • We can also call class methods using interpolation.

    • Things can't do with Interpolation

  • Assingments are not possible inside the expression eg: a = 2+5
  • can't able Acessing Global Variables (windows/screens) bcoz template not aware of global javascript variables
