About Template Reference Variable:

  • Flow data from view to class and To easly access the DOM elements and its properties

  • The hash symbol (#) is used to declare a template reference variable. eg: #VariableName

  • The ref- prefix is an alternative to the hash symbol.

  • It creates a reference to the input element that can be used later.

  • Template reference variables will works with DOM elements, components and directives.

  • Template reference variables are hoisted and can be accessed anywhere in the template.
